# Projeto feito para atividade da faculdade

Este repositório contém o código-fonte do site institucional da **ONG Esperança**, uma organização sem fins lucrativos dedicada à promoção de projetos sociais nas áreas de educação, saúde e desenvolvimento comunitário.

O projeto é composto por três páginas principais:

- **Página Inicial** (`index.html`)
- **Projetos Sociais** (`projetos.html`)
- **Cadastro de Voluntários/Doadores** (`cadastro.html`)
